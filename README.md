# FGVC Aircraft Classification

Fine-Grained Aircraft Classification using Deep Learning and Computer Vision.

## Project Overview

This project compares two approaches for aircraft image classification:

1. Custom CNN built entirely from scratch.
2. ResNet50 Transfer Learning model.

## Dataset

FGVC Aircraft Dataset

- 100 aircraft classes
- ~10,000 images
- Fine-grained image classification task

## Models

### Custom CNN
- Built from scratch
- Batch Normalization
- Dropout
- Global Average Pooling

### ResNet50 Transfer Learning
- Pretrained ResNet50 backbone
- Fine-tuned classifier head
